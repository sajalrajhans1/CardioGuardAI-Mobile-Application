❤️ CardioGuardAI
AI-Powered Cardiac Risk Intelligence for Rural Healthcare

CardioGuardAI is a real-time cardiac monitoring system designed to help patients and caregivers detect early signs of cardiac distress. The platform continuously monitors ECG signals and vital signs, applies machine learning–based risk analysis, and provides instant alerts and first-aid guidance during panic or cardiac events.

Our goal is to enable early intervention and reduce cardiac deaths, especially in rural or remote areas where immediate medical assistance may not always be available.

🚀 Features
📱 Patient Side

Live ECG Monitoring – Continuous real-time ECG waveform visualization.

Vital Signs Tracking – Displays BPM (heart rate), HRV, and SpO₂.

Real-time Anomaly Detection – Detects irregular ECG patterns instantly.

Threat Level Classification – ML model classifies patient state as:

Normal

Suspicious

Critical

Emergency Alerts – Automatically alerts caregivers when risk levels increase.

First Aid Guidance – Provides exercises and breathing techniques during panic attacks or cardiac discomfort.

👩‍⚕️ Caretaker Side

Patient Monitoring Dashboard – Caretakers can monitor patient health status.

Live ECG Stream – Real-time waveform monitoring.

Critical Alerts – Immediate notification if patient condition becomes suspicious or critical.

Vital Stats Overview – BPM, HRV, SpO₂ trends for quick assessment.

🏥 Hardware Integration

The system integrates wearable sensors with microcontrollers for real-time data collection.

Sensors Used

AD8232 ECG Sensor – ECG waveform monitoring

MAX30102 – Heart rate & SpO₂

Temperature Sensor

Motion Sensor

Microcontroller

ESP32

The ESP32 transmits sensor data to the application for processing and visualization.

📡 System Architecture

Sensors collect real-time physiological data.

ESP32 streams the data to the application.

Backend processes ECG signals.

ML model analyzes anomalies and risk levels.

Alerts and recommendations are generated for patients and caretakers.
